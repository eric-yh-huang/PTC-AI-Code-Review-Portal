Execution Sequence

01 Trigger Event
Azure Pipeline 呼叫 Tekton Webhook 並傳遞 PR ID。
02 Gcloud Auth
透過 key.json 認證並獲取 K8s 集群控制權。
03 Tekton PipelineRun
建立 Pods 並執行 module-ord-validate-1。

使用 tkn 指令將 K8s 日誌同步回傳至 Azure DevOps 介面。
